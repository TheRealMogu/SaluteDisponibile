### 📲 3. **Notifiche**

#### ✅ WhatsApp (PRIORITARIO)

Usi **WhatsApp Business Cloud API**:

* Crea un'app Facebook Developer
* Verifica numero e business (gratis)
* Ottieni **Access Token** e `phone_number_id`

#### Invio notifica

```python
import requests

def send_whatsapp(user_phone, message):
    url = f"https://graph.facebook.com/v19.0/{PHONE_NUMBER_ID}/messages"
    headers = {
        "Authorization": f"Bearer {ACCESS_TOKEN}",
        "Content-Type": "application/json"
    }
    payload = {
        "messaging_product": "whatsapp",
        "to": user_phone,
        "type": "text",
        "text": {"body": message}
    }
    response = requests.post(url, headers=headers, json=payload)
    return response.status_code
```

📌 L’utente ti autorizza cliccando un link tipo:
`https://wa.me/393123456789?text=Autorizzo%20le%20notifiche`

📌 Salva il numero solo dopo che l’ha usato.

#### ✅ Email (secondaria)

* `smtplib` per Gmail
* Oppure `Mailgun` (free 500 mail/die)
* SMTP Gmail:

```python
import smtplib
from email.mime.text import MIMEText

def send_email(to, subject, body):
    msg = MIMEText(body)
    msg["From"] = "tuaemail@gmail.com"
    msg["To"] = to
    msg["Subject"] = subject
    with smtplib.SMTP_SSL("smtp.gmail.com", 465) as server:
        server.login("tuaemail@gmail.com", "APP_PASSWORD")
        server.send_message(msg)
```

### 🌐 5. **Pubblicazione su Replit**

#### Come procedere:

1. **Crea progetto Full Stack** su Replit

   * Frontend: `React + Vite + Tailwind`
   * Backend: `FastAPI` in Python
   * DB SQLite via file `.db` in root
2. **Collega frontend e backend** tramite API interne

   * `/api/register`, `/api/notify`, `/api/check`
3. **Cron job di monitoraggio**

   * Usa `apscheduler` o Replit scheduler per scraping ogni 10-15 minuti
4. **Connetti dominio**

   * Compra `salutedisponibile.it` su Namecheap o Cloudflare
   * Punta DNS su Replit (usando IP o CNAME)
